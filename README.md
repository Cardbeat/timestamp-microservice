#### Goals:

1. User can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
2. If it does, it returns both the Unix timestamp and the natural language form of that date.
3. If it does not contain a date or Unix timestamp, it returns null for those properties.


### If you wanna run this project locally just follow these steps: 


#### clone this repo:
```
git clone https://github.com/Cardbeat/timestamp-microservice.git
```

#### install the node modules:

```
npm install
```

#### run the server:

```
npm start
```
